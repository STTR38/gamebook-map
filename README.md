# Legend

## Add Points
Left-click on the canvas to add a point

## Edit points
Right-click on a point to edit it

### Name
Modify the point name, it wil show on the map

### Notes
Add notes that help you out

### Shape
Circle

Square

Triangle

### Monsters
Add monsters with 

HP

ATK

### Inventory
Add the items you found

## Link Points
Click on two points to link them

### Edit link
Right-click on a link to modify it

### Path type
Select a path type :

Normal type - usually when it's specific actions like fighting or looking in a room

Cave - Texture is borwn and curvy

Water - texture is blue and wavy with bubbles

Dirt Path - irregular and light brown

Stairs - Angular and gray

### Color for normal path
Blue - Default

Green - Can help show success

Red - Can help show failure

### Curve mode
Click on "ligne courbe"

## Drag points
Enables dragging and modify the curve of links

## Delete
Click on point to erase it, be careful with overlapping points, drag them first !

Right-click on link to select the delete feature

## Clear All
Resets the map

## Save
Saves the map to your downloads

## Load
Select a save file to keep editing it

## Search
Enables searching for text anywhere

Points will appear in yellow if the text is present
